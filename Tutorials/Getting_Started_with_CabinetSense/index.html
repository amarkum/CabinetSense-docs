    Getting Started with CabinetSense

    Summary

    This guide walks you through the essential steps to get up and running with CabinetSense. CabinetSense is a plugin for SketchUp that turns it into a complete cabinet design, engineering, and manufacturing solution. Whether you are a one-person shop or a large manufacturer, CabinetSense handles everything from 3D design to CNC-ready output. CabinetSense works with SketchUp 2021 and later, on both Windows and macOS.

    How to

    Installation: Download the CabinetSense installer from the CabinetSense website or through the Help menu if you already have an older version installed. Run the installer and follow the prompts. CabinetSense files are placed in your SketchUp Plugins folder. On macOS: ~/Library/Application Support/SketchUp [version]/SketchUp/Plugins/CabinetSense On Windows: C:\Users[username]\AppData\Roaming\SketchUp\SketchUp [version]\SketchUp\Plugins\CabinetSense First Launch: Open SketchUp. CabinetSense loads automatically and shows a start panel. Enter your CabinetSense account email to activate your licence. The start panel shows your plan type, version, and build number. CabinetSense toolbars appear: CabinetSense (main), CS Model, CS Component, CS Multi Component, CS Document, CS Tags, CS Presets, CS Wall Tool, and CS Profile. Setting Up Preferences: Go to CabinetSense > Model > Preferences... Set your Language if not English. Set the Database Folder to where your machining database is stored. Set the Materials Folder to where your .skm material files live. Choose your Primary Units (SketchUp Model Units, Inches, or Millimeters). Enable Scene Management if you want automatic scene creation for each room. Placing Your First Cabinet: Open or create a new SketchUp model. From the SketchUp Components panel, browse the CabinetSense component library. Drag a cabinet component into your model. Click on the cabinet to select it, then go to CabinetSense > Component > Component Options to configure its dimensions, materials, and settings. Assigning Materials: After placing cabinets, go to CabinetSense > Model > Job Materials. Assign shop materials to each material type in your model. Set actual thicknesses and edge banding. Click Save. Previewing Machining: Select one or more cabinets. Go to CabinetSense > Multi-Component > Preview Machining. CabinetSense shows 3D drill holes, dadoes, and other operations in the model. Use Clear Machining to remove the preview. Exporting: Go to CabinetSense > Model > Export... CabinetSense generates cut lists, DXF files, hardware reports, and more. Files are saved to a folder on your computer.

    FAQ

    Do I need SketchUp Pro? CabinetSense works with both SketchUp Pro and SketchUp Studio. Some features like Send to Layout require SketchUp Pro with Layout. What SketchUp versions are supported? SketchUp 2021 and later. Can I use CabinetSense on both Windows and macOS? Yes. CabinetSense works on both platforms. The main difference is the Machining Database format: .accdb (Microsoft Access) on Windows and .pstore on macOS. How do I get the component library? Go to CabinetSense > Help > "Get the latest Library..." to download the latest component library. Where can I learn more? CabinetSense > Help > Online Help opens the online documentation. CabinetSense > Help > Videos opens the YouTube tutorial channel. CabinetSense > Help > Getting Started Help opens the beginner's guide. Use "Make an Appointment..." to book a live training session.

    Tutorials

    Your First Cabinet Project

    Open SketchUp and create a new model. Set your model units (e.g., inches or millimeters). From the Components panel, find the CabinetSense library. Drag a base cabinet into the model. Select it and open Component Options to set dimensions. Drag an upper cabinet and position it above. Open Job Materials and assign shop materials. Run Export... to generate your cut list.

    Understanding the Toolbars

    CabinetSense (main): Buy..., Make an Appointment..., Contact Support..., Online Help, Tutorials... CS Model: Export..., Preferences..., Cabinet Groups, Job Materials, Review Materials, Verify Model Integrity, Scene Management, Machining Application, Profile Rendering, 3D, Delete Construction Lines. CS Component: Component Options, Vertical/Horizontal Cabinet Openings, Fill..., Resize..., Face Me, Align, Toggle Door Opening, Toggle Number of Doors, Create Connector. CS Multi Component: Change Door/DF/Panel..., Preview Machining, Clear Machining, Choose New Drawer System..., Choose New Handle..., Change Palette..., Change Height..., Redraw, Private Layer, Toggle Closet Parts, Apply Construction Profile, Make Unique, Hide/Unhide, Transform. CS Document: Send to Layout..., Add Outside/Inside Dimensions, Clear Dimensions, Assign Part #'s, Show/Clear Part Numbers, Find Part Number. CS Tags: Individual tag toggles for each part type, plus CabinetSense Tags (all). CS Presets: Style presets (Concept, Design, Construction, Diagram, Presentation) and Tag presets (Carcass, Frontage, Drawer, External, Room Only, Hide Room, Presentation).

    Known Issues

    On first install, CabinetSense may take a moment to initialize. Subsequent loads are faster. If toolbars do not appear, go to SketchUp > View > Toolbars and enable the CabinetSense toolbars. Language files are stored separately. If translations are missing, go to Preferences and reinstall the language pack.
